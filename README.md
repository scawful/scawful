## Active Projects

### [Oracle of Secrets](https://github.com/scawful/oracle-of-secrets)
A total conversion of *The Legend of Zelda: A Link to the Past* written in 65816 ASM. Features significant engine expansions, a custom entity system, and entirely new game mechanics built from the ground up.

### [yaze](https://github.com/scawful/yaze)
Native C++/ImGui editor and debugger for Zelda 3. Integrates `z3dk` and `mesen` components into a unified dev workflow. Runs natively on desktop or in the browser via Emscripten.
[**Try the Web Client**](https://yaze.halext.org)

### [Z3DK (Zelda 3 Development Kit)](https://github.com/scawful/z3dk)
A modern toolchain fork of the **Asar** assembler. Adds a Language Server (LSP), JSON diagnostics, and `z3disasm` for structured disassembly. Serves as the compiler backend for `yaze` and modern IDE extensions.

### [Mesen2-OoS](https://github.com/scawful/mesen2-oos)
A specialized Mesen2 fork focused on instrumentation and automation. Features a Socket API for Python/Node.js control and headless support for CI/agent testing. Includes custom debug views for inspecting OoS engine state.

### [Agentic File System (AFS)](https://github.com/scawful/afs)
An experimental orchestration layer for managing multi-agent swarms and context loops directly within the filesystem. Treats documentation, tools, and memory as mountable context nodes for AI agents to "live" in the repo.
