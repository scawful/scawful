# Building modern tools for retro hardware and complex ROM hacks.

## Active Projects

### [Oracle of Secrets](https://github.com/scawful/oracle-of-secrets)
A total conversion of *The Legend of Zelda: A Link to the Past* written in 65816 ASM. Features significant engine expansions and entirely new game systems.

### [Z3DK (Zelda 3 Development Kit)](https://github.com/scawful/z3dk)
A modern toolchain fork of the **Asar** assembler. Adds a Language Server (LSP), JSON diagnostics, and `z3disasm` for structured disassembly.

### [Mesen2-OoS](https://github.com/scawful/mesen2-oos)
A specialized Mesen2 fork focused on instrumentation and automation. Features a Socket API for Python/Node.js control and headless support for CI/agent testing.

### [yaze](https://github.com/scawful/yaze)
Native C++/ImGui editor and debugger for Zelda 3. Integrates `z3dk` and `mesen` into a unified dev workflow.

### [Agentic File System (AFS)](https://github.com/scawful/afs)
An experimental orchestration layer for managing multi-agent swarms and context loops directly within the filesystem.
